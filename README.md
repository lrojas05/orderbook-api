## Installation

1. Clone the repository:
```bash
git clone https://github.com/lrojas05/orderbook-api.git
cd orderbook-api
```
2. Install dependency
```bash
npm install
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev
```

## Test
